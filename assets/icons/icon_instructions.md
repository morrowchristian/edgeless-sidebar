# Icon Generation Instructions

## Required Icon Sizes

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels  
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Design Guidelines

- **Color**: #34a853 (Google Green) or #1a1a1e (dark background)
- **Symbol**: "E" or a side panel icon
- **Style**: Minimalist, flat design
- **Format**: PNG with transparent background

## Quick Generation (Browser)

Use this URL template with your icon:

```https://ui-avatars.com/api/?name=ES&Background=34a853&color=fff&size=128```

## Manual Generation

1. Create a 128x128 base icon
2. Scale down for smaller sizes
3. Ensure text is readable at all sizes
4. Test in Chrome toolbar
