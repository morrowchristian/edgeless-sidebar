# Changelog

All notable changes to Edgeless Sidebar will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.5.0] - 2024-06-15

Added:

- Calculator widget with full functionality
- Delete button for app icons (hover to reveal)
- Clear all apps button
- Context awareness - sidebar reacts to active tab changes
- Keyboard support for calculator
- Enhanced error handling throughout
- Service worker lifecycle management

Changed:

- Complete codebase cleanup and refactoring
- Improved dark theme consistency
- Better accessibility with ARIA labels
- Enhanced network proxy rules for iframe blocking
- Optimized storage management

Fixed:

- Favicon loading issues with Google service
- Duplicate app pinning prevention
- Widget tab switching transitions
- Calculator display overflow issues
- Service worker reconnection issues

## [0.4.3] - 2024-05-20

Added:

- Initial public release
- Split-screen sidebar with left/right alignment
- App tower with pinning functionality
- Persistent notes with auto-save
- Global keyboard shortcuts
- Iframe de-blocker with declarativeNetRequest
- Offline text avatars for apps
- Base dark theme
